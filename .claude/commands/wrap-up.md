# /project:wrap-up — End of session

**Usage**: `/project:wrap-up`

## Steps:
1. Run `forge build && forge test` - verify green
2. Run `git status` - see all changes
3. Update `.claude/todo.md`:
   - Mark completed items `[x]`
   - Add new items to backlog
   - Add session note under "## Session Notes"
4. Summarize to user: what shipped, what's pending

## Session Note Format:
```
### YYYY-MM-DD
**Changes**: What was done
**Open items**: Anything left unfinished
**Handoff**: Context for next session
```
